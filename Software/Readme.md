# ESP8266 Wi-Fi Path Memorizing Robot

Arduino sketch for an ESP8266-based RC robot that can be controlled over Wi-Fi, record movement commands, and autonomously replay the path in reverse using an L298N motor driver.

## Features
- Web-based Wi-Fi control
- Path recording & reverse replay
- PWM motor speed control
- OTA firmware updates

## Hardware
ESP8266 NodeMCU, L298N Motor Driver, DC Motors

## Libraries
ESP8266WiFi, ESP8266WebServer, ArduinoOTA
